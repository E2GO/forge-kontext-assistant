- `/reports/` - Development reports and fixes documentation
- `AUTO_INSTALL.md` - Automatic installation guide
- `BUGFIX_SET_ITERATION.md` - Bug fixes documentation
- `FLORENCE_TAGS.md` - Florence-2 tags reference
- `IMPLEMENTATION_STATUS.md` - Current implementation status
