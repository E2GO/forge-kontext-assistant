# Kontext Assistant v2.77 - Руководство пользователя

## 🚀 Основные возможности

1. **Автоматический анализ изображений**
   - Florence-2 Base для технического анализа
   - PromptGen v2.0 для генерации тегов и описаний
   - Поддержка до 3 контекстных изображений

2. **Умная генерация промптов для FLUX.1 Kontext**
   - 7 типов задач (замена объектов, стиль, освещение и т.д.)
   - Контекстно-зависимые промпты
   - Оптимизация для FLUX

3. **Оптимизации производительности**
   - FP16 для экономии памяти
   - Автоматическая выгрузка моделей
   - Поддержка RTX 5090

## 📖 Как использовать

### 1. Загрузка изображений
- Загрузите до 3 изображений в Forge FluxKontext Pro
- Изображения автоматически появятся в Kontext Assistant

### 2. Анализ изображений
- Нажмите "Analyze All Images"
- Дождитесь завершения анализа (15-40 секунд)
- Результаты появятся под каждым изображением

### 3. Генерация промпта
- Выберите тип задачи (Task Type)
- Опишите желаемые изменения
- Нажмите "Generate FLUX.1 Kontext Prompt"
- Скопируйте результат в основной промпт

### 4. Управление памятью
В Advanced Options:
- **Auto-unload Models** - автоматическая выгрузка моделей
- **Unload Delay** - задержка перед выгрузкой (60 сек по умолчанию)

## 🔧 Решение проблем

### Ошибки h11 Content-Length
Запустите `clear_all_caches.bat`

### Изображения не загружаются
1. Проверьте что изображения загружены в KontextPro
2. Нажмите кнопку 🔄 для обновления

### Низкая производительность
1. Включите Force CPU Mode для слабых GPU
2. Увеличьте Unload Delay для частой работы

## ⚡ Советы

- Первый анализ после запуска медленнее (загрузка моделей)
- Для экономии VRAM используйте Auto-unload
- Для максимальной скорости отключите Auto-unload
